# 这里是教程合集主页😊

本仓库用于存放各种我自己编写的教程，随缘更新  
本仓库的教程有两种展示方式：  

1. **Markdown**，此类型可以直接点击本页面的跳转链接查看，无需下载
2. **Word**，此类型需要自己下载本仓库中的docx文件，用Word或WPS查看

## 如何下载本仓库的word文档👇

跳转到仓库的[**Word**](/Word/) 文件夹后，在教程的页面找到这个按钮![download](/image/how_to_down.png)下载当前选中的压缩包，下载后解压即可
